# Task 37 – Analyze Cost by Service and Resource in Azure Cost Management

## 🎯 Objective

Use Azure Cost Analysis to identify which services and resources are contributing the most to Azure spending.

---

## 📌 Overview

Azure Cost Analysis allows users to break down spending by:

- Service name
- Resource
- Resource group
- Location
- Subscription

In this task, cost was grouped by **Service Name** and **Resource** to analyze spending distribution.

---

## 🛠️ Steps Performed

### Step 1: Navigate to Cost Analysis
- Login to Azure Portal
- Search for **Cost Management + Billing**
- Click **Cost Management**
- Select **Cost Analysis**

---

### Step 2: Group Cost by Service Name
- Click **Group by**
- Select **Service name**

Observation:
Displayed cost breakdown by Azure services such as:
- Virtual Machines
- Storage
- Networking
- Other services

---

### Step 3: Group Cost by Resource
- Click **Group by**
- Select **Resource**

Observation:
Displayed cost breakdown by individual resources such as:
- Specific Virtual Machines
- Storage Accounts
- Other deployed resources

---

## 🔍 Observations

- Identified which service consumed the highest cost.
- Observed spending distribution across resources.
- Verified actual cost for the selected month.

---

## 📊 Outcome

- Successfully analyzed cost distribution.
- Identified top cost-consuming services.
- Improved understanding of cost visibility tools.
