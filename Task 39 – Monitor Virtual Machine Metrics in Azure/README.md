# Task 39 – Monitor Virtual Machine Metrics in Azure

## 🎯 Objective

Monitor the performance of a Virtual Machine using Azure Monitor Metrics.

---

## 📌 Overview

Azure Monitor collects numerical performance data (metrics) from Azure resources.

In this task, VM performance was monitored using the **Percentage CPU** metric.

---

## 🛠️ Steps Performed

### Step 1: Navigate to Virtual Machine
- Login to Azure Portal
- Search for **Virtual Machines**
- Select the target Virtual Machine

---

### Step 2: Open Metrics
- In the left menu, under **Monitoring**
- Click **Metrics**

---

### Step 3: Select Metric
- Metric Namespace → Virtual Machine Host
- Metric → **Percentage CPU**
- Time Range → Last 1 hour (or preferred range)

---

## 🔍 Observations

- Displayed real-time CPU usage graph.
- Observed CPU spikes and average utilization.
- Verified performance monitoring capability.

---

## 📊 Outcome

- Successfully accessed Azure Monitor Metrics.
- Monitored CPU performance of VM.
- Understood how Azure tracks performance data.
