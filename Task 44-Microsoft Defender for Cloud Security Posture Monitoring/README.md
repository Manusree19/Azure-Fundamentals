# Task 44- Microsoft Defender for Cloud Security Posture Monitoring

## Objective
Explore and monitor the security posture of an Azure subscription using Microsoft Defender for Cloud.

## Services Used
- Microsoft Defender for Cloud
- Azure Subscription

## Steps Performed
1. Logged into the Azure Portal.
2. Navigated to **Microsoft Defender for Cloud**.
3. Opened the **Overview** dashboard.
4. Reviewed security metrics including:
   - Azure Subscriptions
   - Assessed Resources
   - Attack Paths
   - Security Alerts
5. Analyzed the **Security Posture** section.

## Observations
- The Azure subscription was detected successfully.
- No resources were assessed yet because no resources were deployed.
- No security alerts or attack paths were found.

## Result
Successfully explored the Microsoft Defender for Cloud dashboard and understood how Azure monitors cloud security posture.
