# Task-21: Create Azure Migrate Project

## Objective
Create an Azure Migrate project to discover, assess, and migrate servers to Azure cloud.

---

## Prerequisites
- Azure Free Tier account
- Resource Group created
- Access to Azure Portal

---

## Steps

1. Sign in to Azure Portal  
   https://portal.azure.com  

2. In the top search bar, type:  
   `Azure Migrate`  

3. Click on **Azure Migrate**

4. Click on **Create project**

5. Enter project details:

   - Subscription: Select your subscription  
   - Resource Group: Select existing or create new  
     Example: `Azure-Migration-RG`  
   - Project Name:  
     `Azure-Migrate-Project`  
   - Geography:  
     `India`

6. Click **Create**

7. Wait for deployment to complete.

8. Open the Azure Migrate project and verify dashboard shows:
   - Discover
   - Assess
   - Migrate

---

## Expected Output
Azure Migrate project successfully created and ready for migration tasks.
