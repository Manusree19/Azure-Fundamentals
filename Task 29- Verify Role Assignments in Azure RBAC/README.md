# Task 29: Verify Role Assignments in Azure RBAC

## Objective
Verify that RBAC roles have been successfully assigned to users and groups. This ensures that the correct access permissions are applied to Azure resources.

---

## Steps

### Step 1: Login to Azure Portal
- Open https://portal.azure.com
- Sign in with your Azure account credentials

---

### Step 2: Navigate to Resource Groups
- In the search bar, type: Resource groups
- Click Resource groups
- Select your resource group (example: rg-rbactest)

---

### Step 3: Open Access Control (IAM)
- In the left menu, click Access control (IAM)

---

### Step 4: View Role Assignments
- Click Role assignments tab
- Review the list of users, groups, and roles

---

### Step 5: Confirm Assigned Roles

Verify that the following roles are assigned:

- testuser1 → Reader role
- developers group → Contributor role

---
## Result

Verified that role assignments are properly configured and access control is working as expected.

---
## Verification

Successfully confirmed that RBAC roles are correctly assigned to both user and group.
