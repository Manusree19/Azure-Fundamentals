# Task 25: Create User in Microsoft Entra ID

## Objective
Create a new user in Microsoft Entra ID to manage identity and enable role-based access control (RBAC) for Azure resources.

---

## Steps

### Step 1: Login to Azure Portal
- Open https://portal.azure.com
- Sign in with your Azure account credentials

---

### Step 2: Navigate to Microsoft Entra ID
- In the search bar, type: Microsoft Entra ID
- Click on Microsoft Entra ID

---

### Step 3: Open Users Section
- In the left menu, click Users
- Click + New user
- Select Create new user

---

### Step 4: Enter User Details
Fill in the following information:

- User principal name: testuser1
- Name: testuser1
- Password: Auto-generated

- Click Create

---

## Result

Successfully created a new user in Microsoft Entra ID.
