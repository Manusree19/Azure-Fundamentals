# Azure Fundamentals – Hands-on Lab 

This repository contains **50 hands-on tasks performed in Microsoft Azure** to gain practical experience with core cloud services.

The tasks cover important Azure areas such as **Compute, Networking, Storage, Identity, Security, Monitoring, Backup, and Management Tools**.

---

## Key Topics Covered

### Compute
- Virtual Machines
- Virtual Machine Scale Sets
- Azure Container Instances
- Azure Functions
- Azure App Services

### Networking
- Virtual Networks
- Subnets
- Network Security Groups (NSG)
- VPN Gateway
- Azure Load Balancer

### Storage
- Storage Accounts
- Blob Storage
- Azure File Shares
- Queue Storage
- Table Storage
- Lifecycle Management

### Identity and Access
- Microsoft Entra ID
- Users and Security Groups
- Role-Based Access Control (RBAC)
- Service Principals

### Governance and Security
- Azure Policy
- Resource Locks
- Microsoft Defender for Cloud
- Azure Key Vault

### Monitoring and Cost Management
- Azure Monitor
- VM Metrics
- Log Analytics Workspace
- Azure Service Health
- Cost Analysis and Budget Alerts

### Backup and Recovery
- Recovery Services Vault
- Virtual Machine Backup

### Management Tools
- Azure CLI
- Azure PowerShell
- Azure Cloud Shell

- Azure CLI
