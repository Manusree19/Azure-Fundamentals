# Task 45- Store Secrets Securely using Azure Key Vault

## Objective
Securely store sensitive information such as passwords or API keys using Azure Key Vault.

## Services Used
- Azure Key Vault
- Azure Resource Group

## Steps Performed
1. Logged into Azure Portal.
2. Created a new Azure Key Vault.
3. Configured vault name, region, and resource group.
4. Opened the Key Vault and navigated to the Secrets section.
5. Added a new secret (example: DBPassword).

## Result
Sensitive credentials were securely stored in Azure Key Vault.
