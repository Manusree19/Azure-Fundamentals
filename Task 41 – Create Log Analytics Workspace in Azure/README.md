# Task 41 – Create Log Analytics Workspace in Azure

## 🎯 Objective

Create a Log Analytics Workspace to collect, store, and analyze log data from Azure resources.

---

## 📌 Overview

Log Analytics Workspace is a central repository where Azure logs are stored and analyzed.

It enables:
- Log querying
- Troubleshooting
- Security investigation
- Performance analysis

---

## 🛠️ Steps Performed

### Step 1: Navigate to Log Analytics Workspaces
- Login to Azure Portal
- Search for **Log Analytics Workspaces**
- Click **Create**

---

### Step 2: Configure Basic Settings
- Subscription → Azure subscription 1
- Resource Group → rg-demo
- Workspace Name → monitoring-workspace
- Region → Central India (or same as VM region)

Click **Review + Create**

---

### Step 3: Create Workspace
- Click **Create**
- Wait for deployment to complete

---

## 🔍 Observations

- Workspace deployed successfully.
- A central log storage resource was created.
- Ready to collect logs from Azure resources.

---

## 📊 Outcome

- Successfully created Log Analytics Workspace.
- Established centralized log storage.
- Prepared environment for advanced log analysis.
