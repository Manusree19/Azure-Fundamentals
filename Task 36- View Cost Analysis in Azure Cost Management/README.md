# Task 36 – View Cost Analysis in Azure Cost Management

## 🎯 Objective

Access and analyze current Azure spending using the Cost Management Dashboard.

---

## 📌 Overview

Azure Cost Management provides visibility into actual Azure spending.

In this task, we explored the **Cost Analysis** section to monitor current usage and spending trends.

---

## 🛠️ Steps Performed

### Step 1: Navigate to Cost Management
- Login to Azure Portal
- Search for **Cost Management + Billing**
- Click on **Cost Management**

---

### Step 2: Open Cost Analysis
- Click **Cost Analysis**
- Select appropriate:
  - Subscription
  - Time range (Current month)

---

## 🔍 Observations

The dashboard displayed:

- Total cost for the selected time period
- Daily spending trend graph
- Cost breakdown by service
- Forecasted cost for the month

---



## 🧠 Key Learnings

- Cost Analysis shows actual Azure spending.
- It helps identify high-cost resources.
- Forecast feature predicts month-end cost.
- Cost can be grouped and filtered for deeper analysis.
